# Pingpong - Social Media

## Core Features

* Post creation
* Chat (one-on-one chat, group chat) with AI chatbot integration
* Video calling
* Notifications
* Admin management
* Groups / Pages

## Security & Authentication

* Email verification required for registration
* Password reset functionality
* Sign up / Sign in with Google, Facebook, and GitHub
* Rate limiting configuration

## Tech Stack

### Frontend

* React TypeScript, TailwindCSS, tanstack/react-query, react/router, zustand, shadcn, daisyUI, HeroUI

### Backend

* NestJS, Prisma, PostgreSQL, Swagger

### Entity Relationship Diagram

* Lucidchart
